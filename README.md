# GEOMETRIA-DINAMICA-COM-PYTHON
Software educacional desenvolvido em Python para auxiliar a compreensão de propriedades geométricas por meio da manipulação dinâmica de figuras planas.

## Descrição

Este projeto consiste em um software educacional desenvolvido em Python com o objetivo de auxiliar a compreensão de conceitos de geometria plana por meio da manipulação dinâmica de figuras geométricas em um plano cartesiano.

A aplicação permite que o usuário interaja com diferentes figuras geométricas, alterando suas dimensões e posição, enquanto visualiza automaticamente propriedades matemáticas importantes, como área, perímetro, vértices e fórmulas utilizadas nos cálculos.

## Objetivos

* Facilitar o aprendizado de geometria plana de forma visual e interativa.
* Relacionar conceitos geométricos com suas representações gráficas.
* Apresentar o cálculo de áreas e perímetros de maneira dinâmica.
* Apoiar atividades educacionais em matemática.

## Funcionalidades

* Visualização dinâmica de figuras geométricas.
* Manipulação da posição das figuras no plano cartesiano.
* Cálculo automático de área e perímetro.
* Exibição das fórmulas matemáticas utilizadas.
* Identificação dos vértices das figuras.
* Atualização em tempo real dos resultados.

## Figuras Disponíveis

* Triângulo
* Quadrado
* Retângulo
* Trapézio
* Circunferência

## Tecnologias Utilizadas

* Python
* NumPy
* Matplotlib
* ipywidgets
* IPython

## Como Executar

### 1. Instale as dependências

pip install numpy matplotlib ipywidgets

### 2. Execute o projeto

O código foi desenvolvido para execução em ambiente Jupyter Notebook ou Google Colab, onde os componentes interativos são suportados.

## Estrutura do Projeto

O sistema é composto por:

* Controles interativos para manipulação das figuras.
* Rotinas de desenho utilizando Matplotlib.
* Cálculo automático das propriedades geométricas.
* Exibição das fórmulas matemáticas correspondentes.
* Interface gráfica baseada em ipywidgets.

## Aplicação Educacional

O software pode ser utilizado como ferramenta de apoio ao ensino de matemática, permitindo que estudantes explorem visualmente propriedades geométricas e compreendam a relação entre fórmulas, medidas e representações gráficas.
